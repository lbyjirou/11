## 四、常见问题排查

### 4.1 提示「登录已过期」

**原因**：Token 过期或无效

**解决**：
1. 清除小程序缓存
2. 重新登录

### 4.2 规格下拉列表为空

**原因**：后端 API 返回空数据，且本地数据加载失败

**解决**：
1. 检查 `utils/calc.js` 是否存在
2. 检查控制台是否有 JS 错误

### 4.3 导出功能报错

**原因**：后端导出接口异常

**排查步骤**：
1. 检查后端控制台日志
2. 在 Knife4j 中测试 `POST /quote/export` 接口
3. 确认 `QuoteCalculateDTO` 数据格式正确

### 4.4 铝价显示为 0

**原因**：后端未配置铝价

**解决**：
1. 在 Knife4j 中调用 `PUT /admin/config/aluminum-price?price=20.5`
2. 重新进入页面
