# 冷凝器报价系统 - 数据库说明

## 数据库信息
- **数据库名**：condenser_db
- **字符集**：utf8mb4
- **用途**：存储冷凝器零部件规格和报价数据

---

## 表结构说明

### 表1：aluminum_price（铝价表）

存储每日铝锭价格和相关参数。

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | INT | 主键，自增 |
| price | DECIMAL(10,2) | 铝锭价（元/KG） |
| diff_ratio | DECIMAL(5,3) | 理论实际差异比，默认1.0 |
| loss_ratio | DECIMAL(5,3) | 损耗比，默认1.02 |
| created_at | DATETIME | 创建时间 |

---

### 表2：collector_spec（集流管规格表）

存储集流管的预设规格参数。

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | INT | 主键，自增 |
| name | VARCHAR(50) | 规格名称，如 16x1.1、20x1.15 |
| area | DECIMAL(10,4) | 截面积（mm²） |
| length | DECIMAL(10,2) | 默认长度（mm） |
| fee | DECIMAL(10,2) | 原料加工费（元），默认16.5 |

**计算公式**：
```
单根重量(kg) = 截面积 × 长度 × 差异比 × 损耗比 × 2.75 / 1000000
单根价格(元) = 重量 × (铝价 + 加工费)
```

---

### 表3：fin_spec（翅片规格表）

存储翅片的预设规格参数。

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | INT | 主键，自增 |
| width | DECIMAL(10,2) | 宽度（mm） |
| wave_height | DECIMAL(10,2) | 波高（mm） |
| pitch | DECIMAL(10,2) | 波距（mm） |
| wave_len | DECIMAL(10,2) | 单波长（mm） |
| center | DECIMAL(10,2) | 默认中心距（mm） |
| wave_count | INT | 默认波数 |
| fee | DECIMAL(10,2) | 原料加工费（元），默认7.0 |
| part_fee | DECIMAL(10,4) | 部件加工费（元），默认0.001 |

**计算公式**：
```
总波长(mm) = 单波长 × 波数
单根重量(kg) = 宽度 × 总波长 × 厚度 × 2.75 / 1000000
单根价格(元) = 重量 × (铝价 + 原料加工费) + 部件加工费
```

---

### 表4：tube_spec（扁管规格表）

存储扁管的预设规格参数。

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | INT | 主键，自增 |
| width | DECIMAL(10,2) | 宽度（mm） |
| thickness | DECIMAL(10,2) | 厚度（mm） |
| length | DECIMAL(10,2) | 默认长度（mm） |
| meter_weight | DECIMAL(10,4) | 米重（kg/m） |
| fee | DECIMAL(10,3) | 普通加工费（元），默认7.436 |
| zinc_fee | DECIMAL(10,2) | 喷锌加工费（元），默认11.9 |

**计算公式**：
```
单根重量(kg) = 米重 × 长度 / 1000
普通单根价格(元) = 重量 × (铝价 + 加工费)
喷锌单根价格(元) = 重量 × (铝价 + 喷锌加工费)
```

---

### 表5：quote_record（报价记录表）

存储历史报价记录，用于导出PDF。

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | INT | 主键，自增 |
| al_price | DECIMAL(10,2) | 当时铝价（元/KG） |
| collector_data | JSON | 集流管数据 |
| fin_data | JSON | 翅片数据 |
| tube_data | JSON | 扁管数据 |
| material_cost | DECIMAL(10,2) | 材料成本（元） |
| mfg_cost | DECIMAL(10,2) | 制造费用（元） |
| profit | DECIMAL(10,2) | 利润（元） |
| freight | DECIMAL(10,2) | 运费（元） |
| final_price | DECIMAL(10,2) | 最终报价（元） |
| created_at | DATETIME | 创建时间 |

**汇总计算公式**：
```
材料成本 = 集流管单价×数量 + 翅片单价×数量 + 扁管单价×数量
利润 = (材料成本 + 制造费用) × 10%
最终报价 = 材料成本 + 制造费用 + 利润 + 运费
```

---

## 常量说明

| 常量 | 值 | 说明 |
|------|-----|------|
| 铝密度 | 2.75 g/cm³ | 用于计算重量 |
| 理论实际差异比 | 1.0 | 集流管计算使用 |
| 损耗比 | 1.02 | 集流管计算使用 |
| 利润率 | 10% | 固定利润率 |
